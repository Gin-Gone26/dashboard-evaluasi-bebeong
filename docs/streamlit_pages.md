# Rancangan Halaman Streamlit

## `app.py`

Halaman publik untuk ASN:

- Informasi ringkas penelitian.
- Form biodata responden.
- Form kuesioner TAM.
- Tombol kirim jawaban.
- Login admin di sidebar.

## `pages/1_Dashboard_Admin.py`

- Statistik jumlah responden.
- Statistik jumlah jawaban.
- Rata-rata PEOU, PU, dan BI.
- Grafik rata-rata variabel TAM.
- Grafik distribusi jawaban Likert.

## `pages/2_Data_Responden.py`

- Tabel data responden.
- Filter data.
- Edit data responden.
- Hapus data responden.
- Export CSV dan Excel.

## `pages/3_Hasil_Kuesioner.py`

- Tabel hasil jawaban kuesioner.
- Filter data.
- Export CSV dan Excel.
- Ringkasan rata-rata variabel.

## `pages/4_Upload_Jamovi.py`

- Upload file hasil analisis Jamovi.
- Tabel riwayat upload.
- Download file hasil analisis.

## `pages/5_Laporan_Evaluasi.py`

- Laporan ringkas evaluasi.
- Statistik umum.
- Rata-rata variabel TAM.
- Unduh laporan dalam format HTML.
